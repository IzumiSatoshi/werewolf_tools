# werewolf_tools
人狼をやるときに楽をするためのツールなど。

## ジャッジメント用人狼推理補助ツールコピペブックマークレット
人狼推理補助ツール（https://jinrou.tsukuyo-oka.jp/ ）で取った占い結果などのメモを、人狼ジャッジメントのチャットに貼り付けれるようにする形式の文字列にしてクリップボードにコピーする。  
クリップボードにコピーするのが上手いかないので、何回か押す必要がある。
